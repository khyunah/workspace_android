Fragment
- 하나의 액티비티가 여러개의 화면( 뷰 컴포넌트 )을 가지도록 만들기 위해 사용하는 개념

사용방법
- XML 파일에서 ViewComponent 로 추가할 수 있다.
- 코드로 동적으로 추가한다. ( 자바 )
    : 추가하고 싶을때 추가를 하고, 제거하고 싶을때 제거 할수 있다.

중요 !
라이프 사이클이 존재한다. ( 액티비티와 마찬가지로 생명주기가 있다. )
액티비티에 종속적이다.



종속적이고 단독으로는 존재할 수 없다.
하나의 액티비티안에 파편 조각이다.
조각 자체만으로 수명주기를 가지는 뷰를 만들어 낼 수 있다.
파편 여러개 만들수 있음.
단독으로 존재할 수 없어 액티비티가 있어야 존재 가능하다. 종속적
파편을 만들어 놓으면 A 액티비티에서 사용하고, B 액티비티에서 사용한다.
재사용성이 높다.
